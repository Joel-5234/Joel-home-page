# Project Goals

## Primary Goals
1. Create a professional personal website for Joel Lopez
2. Support High Tech High School admissions application
3. Showcase Joel's projects, skills, and interests
4. Provide a platform for future project updates

## Technical Goals
- Build with pure HTML, CSS, and JavaScript (no frameworks)
- Ensure responsive design for all devices
- Maintain clean, readable, well-commented code
- Follow semantic HTML best practices
- Ensure accessibility and good contrast

## Design Goals
- Create a modern, clean aesthetic
- Use High Tech High colors (black and gold)
- Make the site easy to navigate
- Ensure fast loading times
- Make content easy to read and understand

## Learning Goals
- Create code that Joel can understand and modify
- Demonstrate best practices for static websites
- Show how to structure a one-page website
- Provide examples of responsive design
