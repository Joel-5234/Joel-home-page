# Joel Lopez Personal Website

A clean, modern, one-page personal website built with pure HTML, CSS, and vanilla JavaScript.

## Project Overview
This website showcases Joel Lopez's work, skills, and goals for High Tech High School admissions and future projects.

## Tech Stack
- HTML5 (semantic HTML)
- CSS3 (vanilla CSS)
- Vanilla JavaScript
- Reference design: html5up-ethereal template

## File Structure
```
/
├── index.html          # Main HTML file
├── styles.css          # All CSS styles
├── script.js           # All JavaScript functionality
├── docs/               # Documentation
├── tests/              # Test files
├── scripts/            # Development scripts
└── ref/                # Reference materials
```

## Development
This is a static site with no build process. Simply open `index.html` in a browser or use a local server.

## Testing
Run Playwright tests from the `tests/ui/` directory.

## Deployment
The site is designed to be hosted on GitHub Pages. Simply push the files to a GitHub repository and enable Pages in the repository settings.
