# Product Requirements Document: Joel Lopez Personal Website

## Overview
A clean, modern, one-page personal website for Joel Lopez, an 8th grade student, to showcase his work for High Tech High School admissions and future projects.

## Goals
- Showcase Joel's projects and skills
- Support High Tech High School admissions application
- Provide a platform for future project updates
- Demonstrate technical and creative abilities

## Technical Requirements
- Pure HTML, CSS, and vanilla JavaScript
- No build tools or frameworks
- Hosted on GitHub Pages
- Three main files: index.html, styles.css, script.js
- Responsive design (desktop, tablet, phone)
- Semantic HTML structure

## Design Requirements
- Light background with dark text
- Black and gold accent colors (High Tech High colors)
- Clean sans serif font stack
- Consistent spacing and white space
- Card-based layout with rounded corners and subtle shadows
- Smooth scrolling navigation
- Hover animations on interactive elements

## Content Sections
1. Header/Hero Section
2. About Section
3. Majors and Goals Section
4. Projects Section (3 projects)
5. Skills Section
6. Video Section
7. Contact Section

## Success Criteria
- Site loads quickly and works on all devices
- All navigation links work correctly
- Content is clear and accessible
- Design is modern and professional
- Code is well-commented and maintainable
